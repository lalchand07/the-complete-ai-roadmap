## 🎯 How to Use This Roadmap

You should learn AI through **two parallel tracks**:

| Track | Focus | Outcome |
|-------|-------|---------|
| **Track 1: Core AI** | Math, ML, Deep Learning, LLMs | Become a real AI engineer/researcher |
| **Track 2: Applied AI** | Prompting, agents, automation | Build useful AI products today |

> 💡 **Recommendation:** Spend ~60% time on Track 1 and ~40% on Track 2 in early stages. They reinforce each other — Track 2 keeps you motivated with quick wins while Track 1 gives you depth.

> ⚡ **Golden rule:** Don't just watch tutorials. **Build something every week**, even tiny things.

## 📚 Prerequisites — What to Know Before Starting

You don't need to be an expert in these, but basic comfort helps.

### 1. Python Programming (most important)
- Variables, loops, functions, classes, lists, dictionaries, NumPy arrays
- 📘 [Python for Everybody by Dr. Chuck](https://www.py4e.com)
- 📘 [CS50P (Harvard)](https://cs50.harvard.edu/python/)

### 2. Math (don't be scared — learn as you go)
- **Linear Algebra** — vectors, matrices, dot products
- **Calculus** — derivatives, gradients (just intuition)
- **Probability & Statistics** — mean, variance, distributions, Bayes
- 📺 [3Blue1Brown — Essence of Linear Algebra & Calculus](https://www.3blue1brown.com)
- 📘 [Khan Academy](https://www.khanacademy.org)

### 3. Command Line & Git
- 📘 [The Missing Semester of Your CS Education (MIT)](https://missing.csail.mit.edu)

> ⏱️ **Don't get stuck here.** Spend 2–4 weeks on Python and basic math, then start AI. You'll fill gaps as you go.

## 🧠 Core AI Concepts You Must Understand

| Concept | What It Means (Simply) |
|---------|------------------------|
| **Artificial Intelligence (AI)** | The broad field of making machines do "smart" things |
| **Machine Learning (ML)** | Algorithms that learn patterns from data |
| **Deep Learning (DL)** | ML using neural networks with many layers |
| **Neural Network** | Layers of "neurons" that transform inputs |
| **Supervised Learning** | Learning from labeled data (input → known output) |
| **Unsupervised Learning** | Finding patterns in unlabeled data (e.g., clustering) |
| **Reinforcement Learning** | Learning by trial and reward |
| **Training / Inference** | Training = teaching. Inference = using the trained model |
| **Model** | The trained file/object that makes predictions |
| **Dataset** | The data you train/test on |
| **Features** | The input variables a model uses |
| **Loss Function** | Measures how wrong the model is — we minimize this |
| **Gradient Descent** | Algorithm that adjusts the model to reduce loss |
| **Overfitting / Underfitting** | Memorizing training data / failing to learn it |
| **Transformer** | The architecture behind ChatGPT, Claude, etc. |
| **LLM** | Large Language Model — huge transformer trained on text |
| **Token** | A chunk of text (roughly a word or sub-word) |
| **Embedding** | A vector representation that captures meaning |
| **Fine-tuning** | Further training a pre-trained model on your data |
| **RAG** | Retrieval-Augmented Generation — giving an LLM external docs |
| **Prompt Engineering** | The craft of writing better LLM instructions |
| **Agent** | An LLM that uses tools and completes multi-step tasks |
| **Hallucination** | When an LLM confidently makes things up |

## 🎓 Track 1 — Core AI / ML / Deep Learning / LLMs

### 📍 Stage 1: Foundations of ML (Months 1–2)

**Goal:** Understand what ML is, train your first models.

- 📘 [Andrew Ng — Machine Learning Specialization](https://www.coursera.org/specializations/machine-learning-introduction) *(Coursera, audit free)*
- 📘 [Kaggle Learn](https://www.kaggle.com/learn) *(free hands-on micro-courses)*
- 📘 [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course)

> 🛠️ **Build:** Train a model to predict house prices on Kaggle.

### 📍 Stage 2: Deep Learning (Months 3–4)

**Goal:** Understand neural networks deeply.

- 📺 [3Blue1Brown — Neural Networks series](https://www.youtube.com/playlist?list=PLZHQObOWTQDNU6R1_67000Dx_ZCJB-3pi)
- 📘 [Andrew Ng — Deep Learning Specialization](https://www.coursera.org/specializations/deep-learning)
- 📘 [fast.ai — Practical Deep Learning for Coders](https://course.fast.ai)
- 📘 [PyTorch Official Tutorials](https://pytorch.org/tutorials/)

> 🛠️ **Build:** Image classifier (cats vs dogs) on Google Colab.

### 📍 Stage 3: Modern Architectures & Transformers (Months 5–6)

**Goal:** Understand how ChatGPT-class models actually work.

- ⭐ [Andrej Karpathy — Neural Networks: Zero to Hero](https://www.youtube.com/playlist?list=PLAqhIrjkxbuWI23v9cThsA9GvCAUhRvKZ) *(THE best free LLM course)*
- 📘 [The Illustrated Transformer (Jay Alammar)](https://jalammar.github.io/illustrated-transformer/)
- 📘 [Hugging Face NLP Course](https://huggingface.co/learn/nlp-course)
- 📺 [Stanford CS224N — NLP with Deep Learning](https://www.youtube.com/playlist?list=PLoROMvodv4rOSH4v6133s9LFPRHjEmbmJ)

> 🛠️ **Build:** Fine-tune a Hugging Face model on a custom dataset.

### 📍 Stage 4: Advanced LLMs, RAG, Fine-Tuning (Months 7–9)

- 📘 [Hugging Face LLM Course](https://huggingface.co/learn/llm-course)
- 📘 [DeepLearning.AI Short Courses](https://www.deeplearning.ai/short-courses/) *(free)*
- 📘 Stanford CS336 — Language Modeling from Scratch

> 🛠️ **Build:** A RAG chatbot answering questions about your PDFs.

## ✅ Your First 7 Days — Concrete Action Plan

- **Day 1:** Sign up for Google Colab, Kaggle, Hugging Face, Claude/ChatGPT
- **Day 2:** Start Andrew Ng's ML Specialization (Course 1, Week 1)
- **Day 3:** Watch 3Blue1Brown's "Essence of Linear Algebra" Ep. 1–4
- **Day 4:** Do Kaggle's "Intro to Programming" + "Python" micro-courses
- **Day 5:** Read Anthropic's prompt engineering guide. Practice 5 prompts on Claude
- **Day 6:** Watch Karpathy's "Intro to Neural Networks and Backpropagation"
- **Day 7:** Pick one beginner project and start it. Push it to GitHub even if incomplete

## 💡 Final Tips

1. **Build > Watch.** Avoid the tutorial trap.
2. **Use Claude/ChatGPT as your tutor.** Ask, debug, quiz.
3. **Start ugly, ship fast.** Your first model will be bad. That's fine.
4. **Pick a niche after 6 months.** NLP? Vision? Agents? RAG? Voice?
5. **Open-source contributions = gold.** Even fixing typos teaches you.
6. **Document your journey.** A GitHub portfolio + blog beats any certificate.
7. **Don't chase every new tool.** Foundations (math, transformers, Python) last.
8. **Math will click later.** Don't wait until you "understand it all."
9. **Consistency beats intensity.** 30 min/day > 5 hours once a week.
10. **Have fun.** The best learners are curious, not stressed.
